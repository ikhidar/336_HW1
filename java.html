About Java

Java is a high-level, class-based, object-oriented programming language designed to be portable across platforms via the Java Virtual Machine (JVM). It emphasizes strong typing and a large standard library.

Typical uses include enterprise applications, Android apps (historically), server-side systems, and large-scale back-end services.
